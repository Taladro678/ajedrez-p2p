# Archivos de Sonido para Ajedrez

Esta carpeta debe contener los archivos de audio para la aplicación de ajedrez.

## Archivos Requeridos

Coloca los siguientes archivos de audio en esta carpeta:

- `move.mp3` - Sonido de movimiento normal
- `capture.mp3` - Sonido de captura de pieza
- `check.mp3` - Sonido de jaque
- `castle.mp3` - Sonido de enroque
- `promote.mp3` - Sonido de promoción de peón
- `game-end.mp3` - Sonido de fin de partida
- `notify.mp3` - Sonido de notificación general

## Fuentes de Sonidos Gratuitos

Puedes descargar sonidos de ajedrez gratuitos de:

1. **Lichess** (código abierto):
   - https://github.com/lichess-org/lila/tree/master/public/sound

2. **Chess.com** (uso personal):
   - Inspecciona la página de Chess.com y descarga desde DevTools

3. **Freesound.org**:
   - https://freesound.org/search/?q=chess

4. **Zapsplat** (registro gratuito):
   - https://www.zapsplat.com/sound-effect-category/chess/

## Formato Recomendado

- **Formato**: MP3 o OGG
- **Duración**: 0.5 - 2 segundos
- **Tamaño**: < 50KB por archivo
- **Bitrate**: 128kbps es suficiente

## Nota

Por ahora, la aplicación funcionará sin sonidos. Los sonidos se reproducirán automáticamente cuando estén disponibles en esta carpeta.
